# Bio Project Management v0.1.0

Initial release for the single-company Bio Project Management platform.

## Highlights

- Vue 3 + Element Plus workspace UI with dashboard, project, task, knowledge, CRM, report, and admin modules.
- NestJS REST API with authentication, users, roles, permissions, translation dictionary, projects, tasks, notifications, communications, knowledge base, dynamic forms, CRM, reports, and search.
- Role-based access control with editable roles and optional direct user permission overrides.
- Project-task synchronization, task-based project progress, comments, notes, approvals, mentions, and in-app notifications.
- Knowledge base with category tree, tags, permissions, rich text, Markdown interoperability, images, and attachments.
- Dynamic form management and CRM integration through the first built-in CRM form.
- Docker Compose deployment with PostgreSQL, Redis, backend, frontend, and persistent uploads.
- Standalone server debug mode with SQL.js for direct Windows server testing without Docker.
